A consultoria é só para quem está endividado? +
Não. Ela serve tanto para quem está endividado quanto para quem quer se organizar melhor e começar a planejar o futuro com mais segurança e clareza.
Preciso entender de finanças para fazer a consultoria? +
De jeito nenhum. A proposta é explicar tudo de forma simples e prática, no seu ritmo, usando uma linguagem que faz sentido para você.
O atendimento é online? +
Sim! A consultoria é realizada totalmente online, o que facilita o acesso independente de onde você esteja.

De forma nenhuma. O objetivo é acolher, organizar e construir um plano possível. Cada pessoa chega com uma história diferente, e todas são bem-vindas.
Quanto tempo leva para ver resultados? +
Muitas clientes relatam mais clareza e menos ansiedade já após a primeira sessão. Os resultados práticos dependem do comprometimento com o plano, mas o caminho fica visível desde o início.
